# HS_Computer_Programming_Logic

So I was rereading Homestuck and somewhere between John ejecting a hammer into his own face and Dave losing a ninja sword because he yelled "stop" at a bird, I realized: this whole thing is a CS lesson that never told me it was a CS lesson.

This repository is me going through every explicitly stated Computer Science concept in Homestuck and actually breaking it down. Not in a "look how smart this comic is" way. More like finding out the thing you have been doing for years has a name, and now you want to know everything about that name.

Everything here is sourced directly from the comic. If Homestuck did not show it on screen or state it in text, it does not live here. That is a separate rabbit hole.

## What is covered

| File | Topic |
|------|-------|
| `docs/01_data_structures_sylladex.md` | Stack, Queue, Array, Binary Tree, Hash Map |
| `docs/02_bitwise_alchemy.md` | AND, OR and how combining items is actually logic gates |
| `docs/03_tilde_ath_language.md` | ~ATH as an esoteric language, loops, imports, bifurcation |
| `docs/04_captchalogue_codes.md` | The math behind 8-character codes and what 64^8 actually means |

## Worked Examples

Each doc above has a companion example that brings the concept to life with step-by-step walkthroughs, character impressions, and interactive challenges.

| File | What Happens |
|------|-------------|
| `docs/examples/01a_john_vs_the_stack.md` | John ejects his entire inventory to reach his PDA. Every push/pop written out |
| `docs/examples/01b_daves_hash_hell.md` | Dave says "stop" to a bird and launches a sword. Full hash math and collision analysis |
| `docs/examples/01c_roses_tree_pruning.md` | Rose removes a non-leaf node and loses half her items. Tree building and balance problems |
| `docs/examples/02a_alchemy_workshop.md` | Bit-by-bit AND/OR on actual codes, decay chains, and punch-card visuals |
| `docs/examples/03a_sollux_code_review.md` | Line-by-line ~ATH breakdowns with Sollux and Karkat commentary |
| `docs/examples/04a_code_space_expedition.md` | The full math of 281 trillion codes, collision probability, and base-64 encoding |

## Who this is for

Anyone who read Homestuck and wants the CS side explained without sitting through a lecture. Or anyone learning CS who wants examples that are not vending machines and grocery lines for once.

No prior CS knowledge required. I explain every term the first time I use it.
